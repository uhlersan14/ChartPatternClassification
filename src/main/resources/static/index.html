About This Project

This application uses deep learning to identify technical analysis patterns in cryptocurrency charts. Upload a chart image, and the application will classify it into one of the supported patterns.

The model is built with DeepJavaLibrary (DJL) and uses a ResNet50 architecture trained on chart pattern examples.
